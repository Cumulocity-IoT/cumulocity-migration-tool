Migration Tool
OVERVIEW

The Migration Tool allows you to migrate web applications, dashboard, groups, devices, simulators, and smart rules from one tenant to another.


After recreating all of the Managed Objects any linkages will also be recreated.

QUICKSTART
  1. Click the Source folder in the navigator
  2. Select Other Tenant, and enter the login details
  3. Select any objects you want to migrate
  4. Click the Destination folder in the navigator
  5. Select Current Tenant, and enter the login details
  6. Click Migrate... in the navigator
  7. Make any edits to the objects
  8. Click Execute Migration in the action bar
